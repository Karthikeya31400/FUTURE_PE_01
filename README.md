Business Chosen
KLG Ancient World – A "Heritage-Tech" student startup developing MythiLens, an AI-powered educational platform that makes history interactive and immersive.

Target Audience
Students, educators, and history enthusiasts (aged 18–45) seeking advanced, gamified ways to explore ancient artifacts and lost languages.

Project Objective
To design a structured, reusable prompt engineering system that generates high-quality, conversion-focused website copy for a technology-driven history platform.

This system produces:

Clear value-driven homepage messaging (The Portal)

Detailed service descriptions for AI features

Strong "Expedition-focused" CTAs

Tone-adaptable website copy

Refined value propositions

Problem Identified
Many history-based educational tools:

Use generic, static, or outdated website copy

Fail to explain complex AI features (Scanning/Translation) clearly

Lack a "portal" or "intersection" experience for the user

Have weak or ineffective booking/launch CTAs

This project addresses those gaps using structured AI prompts.

Prompt Engineering Approach
The system was built using:

Structured business input variables (name, product, features, USP)

Strict output formatting rules

Tone adaptation controls

Benefit-driven copy constraints (PAS Framework)

Value proposition refinement logic

This ensures:

Reusable prompt frameworks for tech startups

Business-specific content generation (Heritage-Tech niche)

Reduced generic AI output

Real-world website readiness
